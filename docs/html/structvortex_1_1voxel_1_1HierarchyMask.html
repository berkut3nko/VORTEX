Detailed Description

Acceleration structure for Ray Tracing (Grid Hierarchy).

Represents a 32x32x32 chunk divided into 8x8x8 blocks (4x4x4 voxels each). Each bit corresponds to one 4x4x4 block. 512 bits total (64 bytes).
