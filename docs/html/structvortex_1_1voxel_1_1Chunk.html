Detailed Description

A single 32x32x32 voxel chunk (SoA Layout).

  • voxelIDs: 8 bits per voxel (Material Index). 32KB.
  • voxelFlags: 2 bits per voxel (State flags). 8KB.
  • hierarchy: 1 bit per 4x4x4 block (Occupancy). 64 Bytes. Total Size: ~40KB per chunk.
