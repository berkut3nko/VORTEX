\doxysubsection{Detailed Description}
The main entry point for the Graphics subsystem. 

Orchestrates the \doxylink{classvortex_1_1graphics_1_1Window}{Window}, Context, \doxylink{classvortex_1_1graphics_1_1Swapchain}{Swapchain}, Pipelines, and Rendering Loop.
