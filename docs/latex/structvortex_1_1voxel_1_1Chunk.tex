\doxysubsection{Detailed Description}
A single 32x32x32 voxel chunk (SoA Layout). 


\begin{DoxyItemize}
\item voxel\+IDs\+: 8 bits per voxel (Material Index). 32KB.
\item voxel\+Flags\+: 2 bits per voxel (State flags). 8KB.
\item hierarchy\+: 1 bit per 4x4x4 block (Occupancy). 64 Bytes. Total Size\+: \texorpdfstring{$\sim$}{\string~}40\+KB per chunk. 
\end{DoxyItemize}
