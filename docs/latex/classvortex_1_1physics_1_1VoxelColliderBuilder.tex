\doxysubsection{Detailed Description}
Responsible for converting Voxel Chunks into optimized physics shapes. 

Uses a Greedy Meshing algorithm tailored for physics boxes (merging voxels into larger boxes). This significantly reduces the number of child shapes in a Jolt Compound\+Shape.
